Teacher Management System 

PURPOSE OF PROJECT:
This project is developed as the second part of our Programming coursework. 
The purpose of this system is to provide an interactive and user-friendly interface for managing teacher and lecturer information in an educational institution. 
It builds upon the core logic implemented in the first coursework by adding a fully functional GUI using Java Swing and AWT.

HOW TO START THIS PROJECT:
Make sure you have Java installed (JDK 8 or higher).
Clone the repository to your local machine.
Open the project in any Java IDE (e.g., IntelliJ, Eclipse, NetBeans).
Compile and run the main class (Main.java or equivalent).
The GUI window will open for you to interact with the Teacher Management System.

USER INSTRUCTIONS:
Use the GUI interface to add, update, or delete teacher records.
The system stores details such as name, subject, and contact information.
Data validation ensures accuracy and completeness of entries.
The system includes testing features to validate functionality and reliability.
All teacher data is managed through an intuitive and clean interface.
